# Bridge AI

Bridge AI is an AI-powered operations platform that helps community organizations reduce repetitive administrative work through Retrieval-Augmented Generation (RAG), document intelligence, and AI-assisted workflows.

## Tech Stack

### Frontend
- Next.js 16
- React
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python

### AI
- Google Gemini (planned)

### Vector Database
- ChromaDB (planned)

## Current Status

✅ Full-stack project foundation completed.
